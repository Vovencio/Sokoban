<p>
A Sokoban-style puzzle game written in pure C for the CASIO-FX-CG series calculators, distributed on <a href="https://www.cemetech.net/downloads/files/2414/x2901">Cemetech</a>. <br>
Ships with 96 levels across four difficulty tiers. This repository contains an earlier build with 8 levels and the Unity-based level designer used to create them.
</p>
